Matrix for your Launchpad

Using Novation's Launchpad for other things than music

Download .zip Download .tar.gz View on GitHub

Control your Novation Launchpad Matrix with your smartphone!

This has to be a dream - I finally finished the script for sending informations to the Launchpad by Novation using the cloud.

SCRIPT WAS NOT CHANGED! DO NOT DOWNLOAD THE PROJECT!

So, what's this about? To get this thing to work, you have to create 3 files in your cloud: 'sccontrol.txt'(without the quotes), 'sendstring.txt'(without the quotes) and 'color.txt'(also without the quotes). You can control the matrix by changing the files in the cloud. Read 'README.md' for more details.

This project has been tested on Launchpad S,

but it should work for Mini and the normal one, too.

This project has been tested on Ubuntu 14.04 Trusty Tahr,

but I think, it should also work on Mac.
